# 🍕 Pizza Sales Analytics Dashboard using SQL & Power BI

## Project Overview

This project analyzes pizza sales data using SQL Server and Power BI to generate business insights and interactive visualizations. The dashboard helps identify sales trends, customer ordering patterns, top-performing pizzas, and key performance indicators (KPIs) for better decision-making.

## Objectives

- Analyze overall sales performance
- Track revenue and order trends
- Identify top and bottom selling pizzas
- Understand sales distribution by category and size
- Create an interactive business intelligence dashboard

## Tools & Technologies

- SQL Server Management Studio (SSMS)
- SQL
- Power BI
- Data Visualization
- Business Intelligence

## Key Performance Indicators (KPIs)

- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Total Orders
- Average Pizzas Per Order

## SQL Analysis Performed

### Revenue Analysis
- Total Revenue
- Revenue by Pizza Category
- Revenue by Pizza Size

### Order Analysis
- Total Orders
- Daily Order Trends
- Monthly Order Trends

### Product Performance Analysis
- Top 5 Pizzas by Revenue
- Bottom 5 Pizzas by Revenue
- Top 5 Pizzas by Quantity Sold
- Bottom 5 Pizzas by Quantity Sold
- Top 5 Pizzas by Total Orders
- Bottom 5 Pizzas by Total Orders

## Dashboard Features

- Interactive Filters
- Revenue Tracking
- Sales Trend Analysis
- Category-wise Analysis
- Size-wise Analysis
- Top and Bottom Product Analysis
- KPI Cards

## Business Insights

- Identified highest revenue-generating pizzas.
- Analyzed customer ordering patterns.
- Determined peak sales periods.
- Compared sales performance across pizza categories and sizes.
- Highlighted low-performing products for improvement.
